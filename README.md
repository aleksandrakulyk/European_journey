# European_journey
## Simple game which helps to choose journey destination.
### This repository uses random module and generates a proposal for next trip destination. User can eliminate country visited lately and answer 5 simple questions to collect personal preferences. The result is a list of countries (or just one country for those less determined), the user may chose to go on a trip.
